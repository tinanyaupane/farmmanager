# Place your app icons here

This directory should contain the following icon files for PWA support:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

You can generate these icons using:
1. Create a 512x512 PNG image with your app logo
2. Use a tool like https://www.pwabuilder.com/imageGenerator to generate all sizes
3. Place them in this folder
